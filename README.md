### 什么是 MCP？

MCP（Model Context Protocol）是 Anthropic 推出的开放标准协议，让 AI 助手能够安全地连接外部数据源和工具。

没有 MCP 时，你需要手动把跑步数据复制给 AI，再把 AI 生成的计划手动录入 App。  
有了跑鸭 MCP，AI 可以：

- **直接读取**你的跑步记录、配速、心率、步频等详细数据
- **直接写入**训练计划，同步到跑鸭 App 并推送到手表
- 无需你在中间搬运数据

### 快速开始

#### 第一步：在跑鸭 App 中开启连接

打开跑鸭 App → **设置** → **连接与授权**，选择以下任意一种方式：

#### 方式一：网页登录授权（推荐）

适合 **Claude、ChatGPT** 等支持打开浏览器授权页的 AI 工具。

1. 复制 MCP 服务地址：`https://yayarun.cn/mcp`
2. 在 AI 工具的 MCP 设置中粘贴服务地址
3. 按提示完成网页登录授权，即可自动建立连接

#### 方式二：手动访问密钥

适合**无法打开浏览器**的工具（如本地 Agent、命令行工具等）。

1. 在跑鸭 App「连接与授权」页点击 **生成密钥**
2. 输入连接名称、选择授权权限和有效期
3. 复制生成的访问密钥（**仅显示一次，请立即保存**）
4. 在 AI 工具配置中填入服务地址和密钥：

```json
{
  "mcpServers": {
    "runyay": {
      "url": "https://yayarun.cn/mcp",
      "headers": {
        "Authorization": "Bearer 你的访问密钥"
      }
    }
  }
}
```

> 每个账号最多创建 5 个手动访问密钥，可随时在 App 中吊销。

### 工具能力一览

连接成功后，AI 可以调用以下工具：

#### 📊 跑步数据查询

**获取最新跑步数据**  
读取你最近一次跑步的完整摘要，包含距离、时长、配速、心率、步频、步幅、功率、最大摄氧量、垂直振幅、温湿度等详细指标。

特别地，跑鸭 MCP 支持获取**分段数据**——即每公里（或每英里）的逐段配速、心率、步频等信息，让 AI 能够深入分析你在一次跑步中的节奏变化、后程掉速情况或体能分配策略。

> 目前高驰（COROS）官方 MCP 暂不支持分段数据查询，跑鸭 MCP 在这一维度具备更细粒度的分析能力。

> 示例提示词：「帮我分析一下我最近这次跑步的分段配速，看看我有没有前快后慢的问题」

**查询历史跑步记录**  
按日期范围查询多次跑步数据（单次最多跨度 90 天），支持按活动类型筛选（户外跑 / 室内跑 / 越野跑）。可用于统计月跑量、分析训练趋势。

> 示例提示词：「统计我这个月的总跑量和平均配速」「对比我上个月和这个月的训练强度」

---

#### 🗓️ 训练计划管理

**查看训练计划**  
查询已有的训练计划列表，可按日期范围、完成状态筛选。

> 示例提示词：「看看我这周还有哪些训练计划没完成」

**推送训练计划**  
让 AI 根据你的目标和近期数据生成结构化训练计划，直接写入跑鸭 App。支持热身、主训练、放松三段式结构，支持间歇跑、有氧跑等多种训练类型，单次最多推送 30 条（可一次安排一周或一个月）。

计划写入跑鸭后，可一键同步到手表执行。**目前已支持同步到佳明（Garmin）、高驰（COROS）和 Apple Watch，其他品牌手表陆续接入中。**

> 示例提示词：  
> 「根据我最近的跑步数据，帮我制定下周的训练计划并推送到跑鸭」  
> 「我想备战半马，帮我制定接下来 4 周的计划，推送到我的佳明手表」

**删除训练计划**  
删除指定训练计划。

> 示例提示词：「把明天的训练计划删掉，我需要休息」

### 常见问题

**Q：数据安全吗？AI 会保存我的跑步数据吗？**  
A：跑鸭 MCP 采用标准 OAuth2 协议授权，数据访问路径为「AI 工具 → 跑鸭官方服务器 → 你的账号数据」，不经过任何第三方。你可以随时在 App 中撤销授权，撤销后 AI 工具立即失去访问权限。AI 工具本身是否留存对话记录，取决于你使用的 AI 产品的隐私政策。

**Q：训练计划推送后会自动同步到手表吗？**  
A：训练计划会先写入跑鸭 App，然后在 App 内操作即可同步到手表。目前已支持**佳明（Garmin）、高驰（COROS）和 Apple Watch**，其他品牌手表陆续接入中。

**Q：访问密钥丢失了怎么办？**  
A：访问密钥仅在创建时显示一次。如果丢失，请在 App「连接与授权」页吊销该密钥，再重新生成一个。

**Q：推送的训练计划能修改吗？**  
A：可以。告诉 AI 你想修改的内容，AI 可以查询到对应计划的 ID，覆盖更新。
